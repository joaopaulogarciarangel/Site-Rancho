Esse é um projeto autoral de software para restaurante, especificamente do meu pai que desenvolvi para otimização da operação da sua churrascaria. Para isso, foi realizado um site para o garçom realizar os pedidos, uma página para a cozinha com cards dos pedidos, e uma página do gerente com funcionamento em tempo real do restaurante. Além disso, na página do gerente tem todo o backoffice do funcionamento da empresa. Adicionei funcionalidades para geração de arquivos csv para futuramente eu fazer a análise de dados e otimizar cada vez mais a operação.
